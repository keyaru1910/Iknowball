# Iknowball

Frontend MVP cho nen tang ty so truc tiep va du doan the thao bang AI.

## Chay local

```bash
npm install
npm run dev
```

Mo `http://127.0.0.1:3000`.

## Build

```bash
npm run build
```

## Cau truc

- `app/page.tsx`: dashboard chinh gom hero, danh sach tran dau, panel AI va thong ke.
- `app/globals.css`: theme dark mode, responsive layout va style thanh du doan.
- `lib/sports-data.ts`: du lieu mock va helper logic cho status/prediction.
- `docs/AI_HANDOFF_NOTES.md`: ghi chu logic tinh nang va cac diem de loi cho AI/dev khac.
